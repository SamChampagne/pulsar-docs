# Pulsar Docs

Site de documentation fonctionnel pour Pulsar, inspiré de l'expérience GitHub Docs: recherche locale, sidebar par sections, pages métier/techniques/API et guide de démarrage.

## Démarrage

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Le site statique est généré dans `dist/`.
